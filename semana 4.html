SEMANA 4 - VIRTUAL

Diseño web: En el diseño web, un sitio web (web site) se compone de una colección de páginas web. Cada página web (web page) es un documento de hipertexto que puede contener diferentes recursos de comunicación, como texto, imágenes, códigos de programa, audio, video, entre otros. Además, las páginas web pueden conectarnos con otros sitios web. Recurso web es una pieza de información usada para transmitir el mensaje deseado.


En el diseño web, la arquitectura se compone de varios elementos importantes. Servidor web o servidor HTTP, que es un programa o conjunto de programas que se ejecutan en un sistema computacional y que esperan conexiones de clientes. Existen diferentes tipos de servidores como Portable web server, pico HTTP server, nginx, Apache, entre otros.


El cliente web o navegador web es el software que se utiliza para obtener, presentar e interactuar con los recursos disponibles en la web. Existen diferentes navegadores como Chrome, Edge, Firefox, Safari, Opera, etc.


La URI o identificador uniforme de recursos es un identificador único de recurso transferible entre el servidor y cliente conformado por una cadena de texto.
